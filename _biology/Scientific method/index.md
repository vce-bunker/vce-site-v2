**"design an experiment" question** (usually 5 marks)
When designing the experiment must show 6 things
How it is done  
What variables are controlled  
Sample size  
Measurements of the IV  
How the DV is measured, interval of it measured/time frame  
Control group  
Repeats

4 mark experiment question
- context
- dependent, independent, 1 control variable (with units)
- control group (if possible)
